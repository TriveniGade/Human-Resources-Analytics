# Human-Resources-Analytics
Project Overview:
This project is an interactive HR Analytics Dashboard built using Power BI for workforce management and HR decision-making.

Features:
-Recruitment analytics
-Attrition analysis
-Employee satisfaction tracking
-Performance analysis
-Hiring forecast

Tools Used:
-Power BI
-Excel
-DAX
-Power Query

Dashboard Pages:
-Recruitment Analytics
-Employee Turnover
-Satisfaction & Performance
-Predictive Analytics

Key Insights:
-Employee attrition trends
-Recruitment performance
-Department-wise turnover
-Workforce growth forecast

Files Included:
-Power BI Dashboard (.pbix)
-HR Dataset
-Dashboard Screenshots
